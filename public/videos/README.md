# Background Videos

Place your video files in this directory.

1.  **Wura Section**: Name your file `wura-bg.mp4`
2.  **Ewa Section**: Name your file `ewa-bg.mp4`

The website is configured to look for these exact filenames.
For best performance, ensure your videos are:
- Compressed (Web Optimized)
- Under 5MB if possible
- Muted (the code mutes them by default)
